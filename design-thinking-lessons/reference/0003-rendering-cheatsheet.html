Rendering Strategy Cheat Sheet

Terms

Server Component
Default component type in the App Router. Runs only on the server, ships no JS of its own to the browser, can access data sources directly.
Client Component
Opt-in via "use client". Needed the moment a component uses state, effects, or event handlers.
use cache
Directive that caches a function's or component's output. The arguments and closed-over values become part of the cache key — get the key wrong and you leak one caller's data into another's response.
Streaming
Sending part of the page immediately (a Suspense fallback) and the rest once it resolves, instead of waiting for everything.
Partial Prerendering (PPR)
A route made of a cached static shell plus dynamic Suspense-wrapped holes, both served from the same request. The current default rendering shape.
