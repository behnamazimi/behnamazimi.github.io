Performance & Scalability Cheat Sheet

Terms

Performance
How fast this is for one user, right now.
Scalability
What happens to cost and latency as load grows. A page can be fast for one user in dev and still fail this axis under production traffic.
Bundle size
Total JavaScript a browser must download and execute before the page is interactive. Grows with every "use client" boundary and its imports.
Hydration
React attaching event handlers and state to server-rendered HTML already in the DOM. Cost scales with client-side tree size, not visible screen area.

Core Web Vitals

MetricMeasuresGood threshold
LCPLoading — time to largest visible content≤ 2.5s
