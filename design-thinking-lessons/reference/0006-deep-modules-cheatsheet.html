Deep Modules Cheat Sheet

Terms

Deep module
Simple interface, hides substantial implementation complexity. High functionality-to-interface-complexity ratio.
Shallow module
Interface complexity is close to the complexity of what it actually does. Little hidden value for the interface cost.
Information hiding
Callers depend on a module's stated promises, never its internals. The mechanism that makes depth possible.
