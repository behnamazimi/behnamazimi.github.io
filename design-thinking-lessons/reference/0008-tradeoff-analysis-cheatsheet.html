Tradeoff Analysis Cheat Sheet

Terms

Consistency
Every read gets the most recent write, or an explicit error — never silently stale data.
Availability
Every request gets a non-error response, without guaranteeing it's the latest write.
Tradeoff analysis
Explicitly naming the specific axes in tension and stating which one is deliberately sacrificed, and why — the opposite of "it depends."

Template for a one-line tradeoff statement

"We chose ___ over ___ because ___." If you can't fill this in for a decision, you haven't finished the analysis yet — you've only picked a side.
